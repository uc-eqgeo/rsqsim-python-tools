rsqsim_api.tsunami.tsunami
==========================

.. py:module:: rsqsim_api.tsunami.tsunami

.. autoapi-nested-parse::

   Classes and functions for loading, manipulating, and plotting sea-surface displacement (SSD) fields.

   Provides :class:`SeaSurfaceDisplacements` for a single event's SSD
   grid, :class:`MultiEventSeaSurface` for a collection of events, and
   :func:`events_from_ssd_netcdf` for extracting SSD arrays from a
   pre-computed netCDF file.

.. py:function:: events_from_ssd_netcdf(event_ids, nc, get_xy = True)

   Extract sea-surface displacement arrays from a netCDF SSD dataset.

   :param event_ids: RSQSim event identifier(s) to extract.
   :type event_ids: int or iterable of int
   :param nc: Open netCDF4 dataset with variables ``"event_id"``,
              ``"ssd"``, ``"x"``, and ``"y"``.
   :type nc: netCDF4.Dataset
   :param get_xy: If ``True`` (default), also return the x and y coordinate
                  arrays.
   :type get_xy: bool, optional

   :returns: * If ``get_xy`` is ``True`` -- x_range : numpy.ndarray
               y_range : numpy.ndarray
               disp_ls : list of numpy.ndarray
             * If ``get_xy`` is ``False`` -- disp_ls : list of numpy.ndarray

   :raises AssertionError: If any requested event ID is not found in the dataset.
