rsqsim_api.tsunami.tsunami

Classes and functions for loading, manipulating, and plotting sea-surface displacement (SSD) fields.

Provides SeaSurfaceDisplacements for a single event’s SSD grid, MultiEventSeaSurface for a collection of events, and events_from_ssd_netcdf() for extracting SSD arrays from a pre-computed netCDF file.
